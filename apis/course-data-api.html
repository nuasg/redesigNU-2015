Courses summary

http://vazzak2.ci.northwestern.edu/courses_summary/

Omits text-heavy fields returned by the Courses endpoint. See below for parameters.

Courses

http://vazzak2.ci.northwestern.edu/courses/

Parameters

Request method 1: term and subject

Parameter Required? Description
term yes The term_id of the term for which you want to see classes.
subject yes The symbol of the subject for which you want to see classes.

Request method 2: term and instructor

Parameter Required? Description
instructor yes The id of the instructor for which you want to see classes.
term no The term_id of the term for which you want to see classes.

Request method 3: find classes using class_num, course_id

Parameter Required? Description
class_num yes One or more class_nums for courses you want details for.
course_id no One or more course_ids for courses you want details for.
term no The term the courses should be in.

Request method 4: get specific classes using id

Parameter Required? Description
id yes One or more ids for courses you want details for.
